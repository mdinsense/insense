Manage Dashboard:


In the Manage Dashboard ,User can do the following operations:
  • View Dashboard
  • Save Dashboard


    • View Dashboard:
  • In View dashboard , user can view the dashboard details for the selected application by selecting the following search Criteria:

      • Environment Name
      Application Name
      Test Login IDs

    Save Dashboard:
  • In Save dashboard , user can save the dashboard details in DB by selecting the below fields:

      • Environment Name
      Application Name
      Test Login IDs
