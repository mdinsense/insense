Manage Skip Urls:


Skip URls are mainly used to inform the scheduler that these URLs should not be skipped by the scheduler.
This page is used to do the following operations:
1.User can add new Skip Urls to the DB by clicking on Add Skip URL button by selecting Environment Name,Application Name & Skip URL
2.User can edit the existing Skip Urls details by clicking on "Edit" button displayed against each Skip URL details in the table
3.User can Delete the Skip Urls details by clicking on "Delete" button displayed against each Skip URL details in the table
4.User can view the Skip Urls details from the table.
