User Administration:


This page is used to do the following operations:

1.User can add new User to the DB by clicking on Add User button by entering UserName,Group Name & Environment Name
2.User can edit the existing user details by clicking on "Edit" button displayed against each user details in the table
3.User can Delete the user details by clicking on "Delete" button displayed against each user details in the table

4.User can view the all existing user details from the table.
