Schedule Testcase:


This page is used to do the following operations:
1.User can add new schedule by clicking on "Submit" button by selecting Application Name,Environment Name,Test case Name,Test Login IDs,Scheduled Time,Recurrence & Skip URLs
2.User can edit the existing schedule details by clicking on "Edit" button displayed against each Schedule details in the table
3.User can Delete the schedule details by clicking on "Delete" button displayed against each Schedule details in the table
4.User can view the schedule details from the table

User can schedule these scheduler application either daily basis or one time basis by selecting or de selecting recurrence check box.

Purpose:
1.Scheduler is used to take the screen shots for selected application-Environment.If there is no baseline,it simply takes the screen shots and if there is a baseline ,it compare the latest screens with baseline screens.
2.After getting all the screen shots reports file will be generated so that user can get the matched & non- matched screens in details.
