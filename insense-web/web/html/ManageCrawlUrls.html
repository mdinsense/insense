Manage Crawl Urls:


Crawl URls are mainly used to inform the scheduler that the scheduler should maintain the domain details specified in crawl URL.It should go to the other domain which is not specified in the Crawl URL.

This page is used to do the following operations:

1.User can add new Crawl Urls to the DB by clicking on Add Crawl URL button by selecting Environment Name,Application Name & Crawl URL
2.User can edit the existing Crawl Urls details by clicking on "Edit" button displayed against each Crawl URL details in the table
3.User can Delete the Crawl Urls details by clicking on "Delete" button displayed against each Crawl URL details in the table
4.User can view the Crawl Urls details from the table.
