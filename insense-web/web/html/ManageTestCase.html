Add Test Case

Using This Screen we can upoad Test case selenium file, whose format is a java. Please Follow Below Steps to Create Selenium Java File using Fire Fox Browser.
Steps
1. Add Selenium IDE Plug in for Fire fox browser.
2. Restart Firefox. After Firefox reboots you will find the Selenium-IDE listed under the Firefox Tools menu.
3. To run the Selenium-IDE, simply select it from the Firefox Tools menu. It opens as follows with an empty script-editing window and a menu for loading, or creating new test cases.
4. Now It is in the state of recording.
5. Proceed Your testing.
6. After completion of testing Stop recording.
7. Save the file (File -> Export test case ->java/JUnit 4/Web Driver)
Now using Add Test Case Screen brows the saved file and upload.
